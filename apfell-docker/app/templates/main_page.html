Welcome to Apfell, where nothing is a bug and everything is a feature

Apfell is meant to be a post-exploit, collaborative, red teaming framework for macOS. Everything is open sourced as well.
Check out my github website for more information on the process of creating Apfell, code repos, and blog posts on updates.

This is the initial, bare-bones release of Apfell that accompanies my Creating an Apfell blog series.

Be sure to go to "Manage Operations"->"Operations Management" and make an operation your "current operation"

Future Updates

  • More Agents
  • More C2 profiles
  • More Analytics
  • Scripting Engine
  • More Services
  • More Reporting Templates
  • More Artifact Tracking
  • Operator Situational Awareness Dashboard
  • Operator Assist/timeout
  • Built-in Keylogging capability in apfell-jxa payload
